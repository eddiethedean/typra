# typra (Python)

Python bindings for **Typra**. **0.2.0** exposes `typra.__version__`; higher-level APIs will ship in later releases.

## Install

```bash
pip install "typra>=0.2.0,<0.3"
```

Supports **CPython 3.9+**. Wheels are published as **`cp39-abi3`** (one per platform for CPython 3.9+).

```python
import typra

print(typra.__version__)
```
